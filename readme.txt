後ろ向き駐車で裏ゴール


2. ゲームのルール

--------------------------------------

ThrillParkingは3Dのドライブシミュレーションゲームです．
車を操縦し規定の場所に止められればクリアですが，燃料と所持金の概念もあり，
操作スキルだけでなく資源の管理も必要となります．

--------------------------------------

3.操作方法

○スタートメニュー画面
  ★説明
      	MainGame: 	基本となるゲームモードです
    	Option  : 	テクスチャの解像度を変更します(一度押すと解像度が下がる)
    	Quit    : 	ゲームを終了します

  ★操作
    矢印キー上下: 	カーソル移動
           zキー:	決定

  ○ゲーム画面
  ★説明
    コースを周回しましょう

  ★操作
      矢印キー上: 	アクセル
      矢印キー下: 	ブレーキ
    矢印キー左右: 	ハンドル移動
           zキー:   	シフトレバー(長押しでリバース)
           xキー:   	ライトon/off切り替え
         escキー:   	ポーズ
